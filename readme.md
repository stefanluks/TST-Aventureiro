# 🎮 TST - Aventureiro

Repositório destinado ao armazenamento de projetos Unity desenvolvidos na disciplina de **Tópicos Especiais em Tecnologia**, com foco em **Desenvolvimento de Jogos 2D**.

---

## 📁 Estrutura

```bash
TST - Aventureiro/
│
├── 3N_O_Aventureiro/
├── 6N_O_Aventureiro/
└── 7M_O_Aventureiro/
````

Cada pasta contém o mesmo projeto base (**Aventureiro**), utilizado por diferentes turmas.

---

## 🎯 Objetivo

Organizar e manter os projetos utilizados em aula, permitindo acompanhamento, versionamento e reutilização do conteúdo entre as turmas.

---

## 🛠️ Tecnologias

* Unity
* C#
* Desenvolvimento 2D

---

## 📌 Observação

Este repositório é de uso educacional e serve como base para as atividades práticas em sala de aula.
